Introduction to CSS

Designing Gotham’s Look

CSS defines how every part of your site looks—colors, fonts, spacing, and more. Just like Batman’s gadgets, CSS gives you complete control over how your website functions and appears.

Why External CSS is the Hero

External CSS files are reusable across multiple pages, making design updates simple and efficient. You can change your entire website’s look by editing one file—just like switching from Bruce Wayne to Batman in seconds.

  • Inline CSS — one element
  • Internal CSS — one page
  • External CSS — entire website
  • Comments keep your code clean
